### install packages
```
npm install -g typescript
```

### compile and execute
```
ts index.ts
node index.js 

### ts-node
  - tsc + node
  - npm install -g ts-node
  - ts-node index.ts

### parcel-bundler
  - tool to help us run typescript in the browser
  - npm install -g parcel-bundler
  - make html with scripts of ts
  - parcel index.html
